## 수강 신청 게임 

### 기능
### 1. 회원 가입
- 학번, 비밀번호, 이름, 학과 입력
- `student_information.md`에 저장
- 중복된 정보 있으면 이미 있는 회원이라고 확인
- 비밀번호 유효성 검사
  - [ ] 7글자 이상
  - [ ] 영문, 숫자 혼용 사용
  - [ ] 비밀번호 재입력과 일치하지 않을 때 다시 제출 허용

### 2. 로그인
- `student_information.md`에 학생 정보 읽어오기
- 입력 시 자동으로 양쪽 공백 제거

3. 강의 목록 명세
- `lecture_information.md`에서 강의 정보 읽어오기

4. 수강신청


5. 종료
